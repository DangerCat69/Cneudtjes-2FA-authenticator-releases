# Cneudtje's-2FA-authenticator-releases

In deze repository staan de publieke builds voor mijn eigen authenticator project genaamd "Cneudtje's Authenticator".
Cneudtje's Authenticator is een Winforms App waarin je zeer gemakkelijk je 6-cijfers code kan kopieëren. 
De "secrets" en bijhorende naam voor wat het dient zijn veilig in een aparte JSON geborgen met encryptie! Je kunt zelf kiezen waar je het bestand bewaard.
Er is ook een login zodat niet iedereen zomaar de app kan starten moesten ze op jouw toestel bezig zijn en dan zo gewoon je 2FA omzeilen.

Dit is iets waaraan ik nog werk en ben mij er bewust van dat er nog redelijk wat werk aan de winkel is.

## Installatie

1. Bij releases download setup.exe
2. Start de installatie door setup.exe te runnen
3. De app wordt dan vanzelf volledig automatisch geïnstalleerd

## Automatische updates

Deze applicatie controleert automatisch op updates bij het opstarten.
Als er een nieuwe versie beschikbaar is, wordt deze automatisch gedownload en geïnstalleerd.

Je hoeft dus niet telkens opnieuw een nieuwe versie te downloaden.
